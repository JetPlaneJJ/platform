To get started:
  1. Sign in (use the button on the top right)
  2. Request access to be an administrator.
If you're having trouble signing in send a message to feedback@mapseed.org and we'll fix it right away!

Once you've been approved as an admin, you'll have the ability to edit certain sites on the map

Make sure you're signed in
Open a site, hit "Edit this post"

If you don't see an "Edit this post" button, you are either not signed in or you don't have permission to edit that particular site.


Hit the Save button
  • You can use the Edit this post button to cancel your changes and go back.
  • Hit the Remove button to remove the site from the map.
  • Save a post in draft mode by setting it from Published to Not published. When a post is in draft mode, it will not show up on the map. Only admins who are signed in can see it.

You can also add new featured sites, directly to the map!

  1. Make sure you're signed in.
  2. Hit "Add a report" as if you were adding a community report.
  3. Select the option to "Add a featured site"

Draw your site on the map

You can add a marker and select a marker symbol, draw a line, or draw a polygon.

Compose your content

The content editor offers you a variety of formatting options such as:

  • Header selection for large or mid-sized page headers
  • Images and video embeds (Youtube or Vimeo)
  • Rich text bold, italics, underline, and color selection
  • Hyperlinks to other web pages

Hit the Save button if you're ready to publish. Otherwise, you can store it as a draft by setting the state to unpublished at the top of the editor.

Have fun creating your community map!
